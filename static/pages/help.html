Frequently Asked Questions

How do I upload files?

expand_more

To upload files to GoFileBeam:

  1. Drag and drop files onto the upload area, or click "Select Files" to choose files from your computer
  2. Choose an expiration option (how long the file will be available)
  3. Optionally, add a password for extra security
  4. Click "Upload" to start the upload process
  5. Once uploaded, you'll receive a shareable link

How do I download files shared with me?

expand_more

To download files shared with you:

  1. Click on the download link provided by the sender
  2. If the file is password-protected, enter the password when prompted
  3. The file will automatically download to your computer
  4. Save the file to your desired location

Note: Some browsers may ask for confirmation before downloading.

What are the file size limits?

expand_more

GoFileBeam has the following limits:

  • Maximum file size: 100 MB per file (configurable by server administrator)
  • Total storage: 1 GB total storage (configurable by server administrator)
  • File types: All file types are supported
  • Multiple files: You can upload multiple files at once

These limits help ensure fast and reliable service for all users.

How does password protection work?

expand_more

When you password-protect a file:

  • The file is encrypted with AES-256-GCM encryption before upload
  • The encryption key is derived from your password
  • We never see your password or the unencrypted file contents
  • To download, recipients must enter the exact same password
  • If the password is lost, the file cannot be recovered

Tip: Use a strong, unique password and share it securely with intended recipients.

How long are files stored?

expand_more

Files are stored based on your chosen expiration option:

  • 1 Download or 1 Day: File expires after 1 download or 24 hours
  • 10 Downloads or 7 Days: File expires after 10 downloads or 7 days
  • 100 Downloads or 30 Days: File expires after 100 downloads or 30 days

Whichever limit is reached first triggers file deletion. Expired files are permanently and securely deleted.

Can I delete files before they expire?

expand_more

Currently, GoFileBeam doesn't provide a way to manually delete files before their expiration date. This is by design to ensure:

  • Files remain available to recipients as promised
  • Consistent behavior for all users
  • Simplified user experience

If you need to prevent access to a file, you can:

  • Ask recipients to delete their copies
  • Stop sharing the download link
  • Wait for the file to expire automatically

Is there a mobile app?

expand_more

GoFileBeam is a web-based service that works on all modern devices:

  • Desktop computers: Full functionality in all modern browsers
  • Tablets: Responsive design works on iPad and Android tablets
  • Smartphones: Mobile-optimized interface for iPhone and Android

You can access GoFileBeam from any device with a web browser. No app installation is required.

What happens if I lose the download link?

expand_more

If you lose a download link:

  • Check your browser history for the link
  • Look for any emails or messages where you might have saved it
  • Ask the sender to share the link again

For security reasons, we cannot:

  • Look up files by filename or sender
  • Provide download links without the file ID
  • Recover lost passwords for encrypted files

Tip: Save important download links in a secure location.
